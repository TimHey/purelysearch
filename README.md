# purelysearch.com

One-page storefront for PurelySearch LLC (Tim Hey's consulting entity).

- Static HTML, no build step. Edit `index.html`, push to `main`, GitHub Pages deploys it.
- Domain: registered at Squarespace, DNS on Cloudflare, served via GitHub Pages (`CNAME` file).
- Email: tim@purelysearch.com via Cloudflare Email Routing → Gmail.
